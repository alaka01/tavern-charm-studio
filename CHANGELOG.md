# Charm ST 更新记录

## v0.4.0 (2026-03-08)

### 新增功能
- **手账风全局预设** — 第 6 套内置风格：粉色卡片气泡、🌸 emoji 头像、渐变粉色状态面板、虚线边框视觉风格、圆形翻页触发按钮
- **正则测试页** — 新增「🧪 正则测试」标签页：左右分栏（输入/预览），深色背景实时渲染正则替换结果，每条脚本显示匹配状态（✓ N 处 / ✗ 未匹配），支持一键加载基于当前角色和状态字段的示例文本
- **纯 CSS 限制说明** — 使用说明页新增黄色警告框，说明 SillyTavern 不执行 JavaScript，所有交互必须用 :checked hack 纯 CSS 实现

### 修复
- 正则测试页替换逻辑统一使用 `g` flag，匹配计数与实际替换结果一致
- 正则脚本构建函数增加 try/catch 防御，单条脚本异常不会导致整个测试页崩溃
- 手账风状态面板 bgColor 从 CSS gradient 改为纯色 `#fff5f5`，兼容 `background-color` 渲染
- 全量预览覆盖层改为 `fixed inset-0 overflow-y-auto`，底部不再透出页面内容
- 正则测试示例文本增加翻页标签（frontTag / backTag / numberTag）包裹
- 预览中进度条根据示例值动态计算百分比，不再写死 60%

## v0.3.0 (2026-03-08)

### 新增功能
- **全量预览覆盖层** — 眼睛图标按钮打开全屏预览，展示对话气泡 + 状态面板 + 翻页效果，点击可翻转正反面
- **格式提示词预览** — 格式提示词标签页支持实时预览生成的提示词文本，可复制或下载
- **滑块实时预览** — 所有标签页中的滑块拖动时预览区域即时更新
- **状态面板示例值双击编辑** — 预览区域双击状态字段值可直接修改示例内容

## v0.2.0 (2026-03-08)

### 新增功能
- **配置持久化** — Zustand persist 中间件，配置自动保存到 localStorage，刷新不丢失
- **设置抽屉** — Header 齿轮按钮打开设置面板，支持导出配置 JSON、导入配置 JSON、重置为默认值（二次确认）
- **配置 UI 简化** — 对话气泡高级选项默认折叠，状态面板新增三个快速模板，翻页卡片标签添加说明注释

## v0.1.0 (2026-03-08)

### 新增功能
- **引导系统** — 全屏欢迎页（原理说明 + 前后对比）→ 三步向导（选风格 → 填角色 → 导出说明）
- **5 套全局风格预设** — 温暖日常、暗黑哥特、赛博朋克、清新校园、极简纯净，一键应用完整配置
- **使用说明页** — 工作原理、4 步导入教程、5 条 FAQ、重新运行向导按钮
- **基础编辑器** — 对话气泡（line/card/quote）、状态面板（分组 + 多布局）、文字特效、翻页卡片、格式提示词生成、导出中心（ZIP/JSON/复制）
